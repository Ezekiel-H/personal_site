Bars per capita in NZ

A 1 minute read, Posted by Ezekiel Haggart on Sat Sep 23, 2017
In Leaflet, R
Tags nz, bars, analysis, shiny

In New Zealand to distribute alcohol you need to apply to your regional Liqour Licence Authority (LLA) for a permit. The NZ Justice Department produces a list of all liqour licences distributed in a single year.

I made this shiny app using a data set from NZ Justice Department and a shapefile/population statistics from Statistics to show for each LLA how many persons per bar, club or liqour retailer. This can be understood as bars per capita for NZ grouped by LLAâ€™s.

Further work planned on this app is to add a drill down function, regional statistics and to account for age with the R18 drinking age.
